## 📌 Proyectos destacados

### 🔧 TheMechanic — Sistema de gestión de talleres (Fullstack)

> API REST + cliente Android para la gestión integral de un taller mecánico. Backend con Spring Boot 4 & Java 21, frontend Android nativo con Kotlin y Jetpack Compose.

**Backend** → arquitectura por feature, JWT con roles ADMIN/MECHANIC/CLIENT, PostgreSQL/H2, Flyway, MapStruct, Swagger, Docker.

**Android** → autenticación con access/refresh token, navegación por roles, UI con Jetpack Compose.

![Java](https://img.shields.io/badge/Java-ED8B00?style=flat&logo=openjdk&logoColor=white)
